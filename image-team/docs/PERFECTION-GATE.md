# PERFECTION-GATE

The system may only be declared "perfect and ready for execution" when all of these are true:

1. A runnable local engine is verified on this machine
2. At least one execution adapter is wired end-to-end
3. A real job produces:
   - image_preview.*
   - image_v1.*
   - image_final.*
   - workflow_notes.txt
   - qa_report.txt
4. The system can preserve originals non-destructively
5. The system can report PARTIAL honestly when likeness is weak
6. At least one smoke test with real output files has passed

Until then, the system is under construction and must not be presented as complete.
