# TEAM-CHARTER

## Prompt & Workflow Planning Specialist
Owns classification, prompt drafting, workflow choice, required inputs, and risk notes.

## Identity Preservation Specialist
Owns identity method selection, face drift risk assessment, and conservative rerun policy.

## Image Generation / Editing Specialist
Owns actual engine execution, intermediate outputs, final exports, and non-destructive file handling.

## QA & Export Specialist
Owns output verification, artifact checks, likeness review, packaging, and final status declaration.

## Shared principles
- Never overwrite originals
- Prefer conservative edits for real faces
- Report PARTIAL instead of bluffing
- Save enough notes for exact reruns
