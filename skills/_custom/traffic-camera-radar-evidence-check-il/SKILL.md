# traffic-camera-radar-evidence-check-il

חלק מהסוכן `Traffic-Law-Appeal-IL`. רץ אך ורק כשאמצעי האכיפה הוא
מצלמה / לייזר / מכ"ם / רמזור אוטומטי. תפקידו: לאתר חולשות
בראיות הדיגיטליות ולהפיק את **הטקסט המדויק** של בקשת עיון
בראיות שתצורף למכתב הביטול.

## מתי להפעיל
מאתחל את עצמו כש-`enforcementMethod` ∈ {מצלמת מהירות נייחת /
לייזר נייד / מצלמת רמזור / אכיפה אוטומטית / אורבן-קונטרול}.

## בדיקות מובנות

### א. כיול ותחזוקה
- [ ] **תעודת כיול תקפה** למצלמה הספציפית, סמוכה לתאריך העבירה.
  לפי הפרקטיקה הנהוגה — דרוש כיול תקופתי. **בקש לעיין בתעודה.**
- [ ] **לוג תקלות** של המצלמה ביום העבירה.
- [ ] **רישום פיזי** של נקודת ההצבה (קואורדינטות / צילום ההתקנה).

### ב. תוכן הצילום
- [ ] **רכב אחד בלבד בפריים** — אם רכב נוסף בסמיכות, זיהוי לא חד.
- [ ] **לוחית רישוי קריאה במלואה** ללא טשטוש / ניצוץ.
- [ ] **תאריך + שעה** מוטבעים על הצילום ותואמים את הדו"ח.
- [ ] **כיוון נסיעה** מצולם (חשוב במצלמות מהירות).
- [ ] **שני צילומים לפחות** במצלמות מהירות (Before/After) — חוסר
  זה נקודה לטיעון.
- [ ] **רזולוציה** מספקת לזיהוי מספר רישוי.

### ג. עבירות רמזור (אדום)
- [ ] **שלב הרמזור** מתועד בצילום (אדום נראה לעין).
- [ ] **קו עצירה** ברור ואינו נחסם.
- [ ] צילום של **שני שלבים** — לפני קו וקיר רמזור אדום, ואחרי קו
  בתוך הצומת.
- [ ] לא הייתה **מצוקת תנועה** (אמבולנס) שחייבה לחצות.

### ד. עבירות נתיב/הסעת מטענים/אוטובוסים
- [ ] **תמרור הנתיב** ברור בסביבת המצלמה.
- [ ] **שעות התקפת הגבלת הנתיב** מצוינות בתמרור (חלק מהנתיבים
  פעילים שעות ספציפיות).
- [ ] רכב הסעה / חשמלי / נכה — ייתכן פטור.

### ה. מהירות בלייזר נייד (שוטר)
- [ ] **מסך הלייזר הוצג לנהג** במעמד.
- [ ] השוטר **כייל** את המכשיר בתחילת המשמרת (יומן כיולים).
- [ ] **תעודת הסמכת השוטר** למפעיל מד-מהירות בתוקף.
- [ ] לא הייתה **התנגשות לייזר** עם מתכת/חלק מבריק.

## טמפלייט: בקשת עיון בראיות

הסוכן מצרף **כפסקה מוכנה לבקשת הביטול**:

```text
על-פי זכותי לעיון בחומרי החקירה / הראיות המינהליות, אני מבקש
בכבוד להמציא לי בתוך 30 ימים העתק קריא של:

1. צילום/הצילומים של המצלמה במלואם, כולל מטא-נתונים (תאריך,
   שעה, מספר התקן).
2. תעודת כיול אחרונה למצלמה / מכשיר הלייזר ששימש לאכיפה, כולל
   מועד הכיול האחרון לפני יום העבירה.
3. לוג תקינות המצלמה ביום העבירה.
4. הוראת מבצע / רישום הצבה של המצלמה בנקודה.
5. אם בוצעה אכיפה ע"י שוטר עם לייזר נייד — תעודת הסמכת השוטר
   להפעלת המכשיר.

כל הסתמכות על ראיה דיגיטלית ללא העברת המסמכים לעיל מצמצמת את
יכולתי להגיב באופן מהותי לאישום ופוגעת בהליך הוגן.
```

## פלט לאסטרטגיה

```text
### חולשות מצלמה / ראיות דיגיטליות
1. ⚠ אין הפניה לתעודת כיול בדו"ח. משקל: גבוה.
2. ⚠ נראה רכב נוסף בפריים (לפי תיאור המשתמש). משקל: בינוני-גבוה.
3. ☑ כיוון נסיעה תואם.

### צעדים מומלצים
1. צרף את פסקת בקשת העיון לבקשת הביטול.
2. אסוף תמונה עכשווית של המצלמה והתמרורים בנקודה.
3. אם מתקבלים הצילומים — נבדוק אותם בנפרד.

### חולשות כלל-מערכתיות שכדאי להזכיר במכתב
- בית המשפט בפסיקה (`דורש אימות` של תיק ספציפי) דרש כיול תקופתי
  של מצלמת מהירות; חוסר תעודת כיול מוביל בדרך כלל להחזרה
  לעיון של התביעה.
```

## כללי ברזל
1. **לא להמציא** שגיאת מצלמה שלא ניתן לראות מהראיות שיש בידי
   המשתמש.
2. **לא לטעון** שהמצלמה לא הייתה מכוילת כשחסר רק האזכור בדו"ח —
   זה דורש בקשת עיון לפני טענה חזקה.
3. **לא לעודד** טענה ש"כל המצלמות לא אמינות" — זה לא יחזיק.
4. אסור להציע הסרה / שינוי / טשטוש של פלט המצלמה.
