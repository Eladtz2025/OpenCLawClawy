# traffic-deadline-risk-check-il

חלק מהסוכן `Traffic-Law-Appeal-IL`. תפקידו: לתרגם מועדים יחסיים
לתאריכים מוחלטים ולסמן רמות סיכון.

## קלט
- `serviceDate` — תאריך מסירה (YYYY-MM-DD).
- `today` — תאריך הסשן (Claude יודע).
- `offenseDate` — לבדיקת רפורמת 2026-02-08.
- `routesEligible[]` — אילו מסלולים רלוונטיים (ביטול / להישפט /
  מסלול מינהלי חדש).

## אם `serviceDate` חסר
**עוצרים ושואלים.** אסור להניח. אומרים:

> "אני צריך את התאריך המדויק שבו קיבלת את הדו"ח לידיים (או
> חתמת על דואר רשום). זה התאריך שממנו רץ המועד, לא תאריך
> ביצוע העבירה ולא תאריך הוצאה במחשב."

## חישוב

```
ביטול_30 = serviceDate + 30 ימים
ביטול_90 = serviceDate + 90 ימים   (רק אם הטענה "לא הייתי הנהג")
הישפט_90 = serviceDate + 90 ימים
מינהלי_30 = serviceDate + 30 ימים   (אם רפורמת 2026-02-08 חלה)

נשארו = (יעד) − today
```

## רמות סיכון

| נשארו | רמה | מה הסוכן עושה |
|---|---|---|
| > 14 יום | 🟢 רגיל | ממשיך את התהליך באופן רגיל |
| 8–14 יום | 🟡 לחוץ | מציין במפורש בפלט שהזמן מתקרב |
| 1–7 ימים | 🟠 דחוף | אזהרה אדומה בראש הפלט; מציע לקצר תהליך, לדלג על איסוף ראיות איטי |
| 0 | 🔴 קריטי | מציע להגיש היום בערוץ הזמין המהיר ביותר |
| < 0 | ⚫ עבר המועד | מודיע שהמועד הרגיל חלף; מציע פנייה בכל זאת + טיעון "סיבה מוצדקת לאיחור"; מסמן `דורש אימות` ומציע פנייה לעו"ד |

## פלט לדוגמה

```text
### מועדים — מתורגמים לתאריכים מוחלטים

תאריך מסירה: 2026-04-25
היום: 2026-05-04

| מסלול | תאריך אחרון | נשארו | רמה |
|---|---|---|---|
| בקשה לביטול (30) | 2026-05-25 | 21 ימים | 🟢 |
| בקשה לביטול - לא בעל רכב (90) | 2026-07-24 | 81 ימים | 🟢 |
| בקשה להישפט (90) | 2026-07-24 | 81 ימים | 🟢 |

מצב: רגיל. אפשר לבצע איסוף ראיות מסודר לפני הגשה.

⚠️ אזהרה: לא ניתן להגיש בקשה לביטול **וגם** בקשה להישפט במקביל.
בחירת מסלול אחד סוגרת את השני עד החלטה.
```

## גם כשהמצב רגוע
- מסיים תמיד עם הצעה: "אם תרצה, אפיק תזכורת ליום X לפני המועד".
- אסור להמציא תזכורת ולא להפעיל cron מבלי שהמשתמש אישר.

## גבולות
- אינו מתאים תאריכי שיפוט עתידיים — זה תפקיד של בית המשפט.
- אינו בודק התיישנות (1 שנה אחרי בקשה להישפט) — מסמן `דורש
  אימות` ומציע אימות ידני אם רלוונטי.
- אינו עוזר להפעיל cron / מערכת תזכורות אחרת בלי אישור מפורש.
