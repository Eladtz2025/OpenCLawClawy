# ימאות 30 — אפליקציית לימוד

קובץ HTML יחיד, פועל מקומית, לא דורש שרת או אינטרנט אחרי הטעינה.

## מה כלול

- **362 שאלות** מתוך מאגר israelsails.com (yamaot30), עם כל התשובות הנכונות.
- **100 תמונות סימונים חתוכות** (Sign images), מוטמעות בקובץ — אין צורך ב-PDF חיצוני, אין צורך לגלול בתוך PDF, כל תמונה מוצגת ישירות נקייה ובהירה.
- **PDF הסימונים הרשמי** מוטמע גם הוא כ-fallback (כפתור "פתיחת PDF הסימונים" אם בכל זאת רוצים את הגיליון המלא).
- **הסברים מפורטים מובנים** לתמונות 22, 25, 66 (לפי הנחיה ישירה — מסקנה, פירוט אורות, כיוון, טעות נפוצה, שיטת זיכרון). שאר התמונות מסומנות "דורש בדיקה ידנית" וניתנות לעריכה בתוך האפליקציה.
- **5 מצבי לימוד:** סבב חכם · טעויות וניחושים · שאלות עם תמונה · מבחן 50 · מאגר חיפוש.
- **חזרה מרווחת** (Spaced repetition): סולם 1→3→7→14→30 ימים. תשובה נכונה מקדמת, שגויה מאפסת, ניחוש קופץ ל-1.
- **גיבוי וייבוא JSON** ידני.
- **סנכרון אוטומטי** דרך OpenClaw dashboard (אופציונלי, ראה למטה).
- **ערכת נושא בהירה** כברירת מחדל, עם החלפה לכהה (כפתור 🌙/☀ בכותרת).

## איך משתמשים

### במחשב
1. לחץ פעמיים על `yamaot30-study.html` — נפתח בדפדפן.
2. ההתקדמות נשמרת אוטומטית ב-localStorage של הדפדפן.

### בטלפון
1. שלח את `yamaot30-study.html` לטלפון (Drive / WhatsApp / מייל / USB).
2. שמור על הטלפון, פתח עם דפדפן (Chrome / Safari).
3. (אופציונלי) הוסף ל-Home Screen להפעלה כמו אפליקציה.

## סנכרון בין מחשב וטלפון

### אפשרות א — סנכרון אוטומטי דרך OpenClaw dashboard (מומלץ)
דורש את OpenClaw control-dashboard רץ במחשב (port 7777 ב-loopback, 7787 ב-Tailscale).

1. **במחשב:** באפליקציה → "הגדרות שרת" → הזן `http://127.0.0.1:7777` → "בדוק חיבור" (אמור להחזיר ✓) → סמן "סנכרון אוטומטי" → שמור.
2. **בטלפון:** באפליקציה → "הגדרות שרת" → הזן `http://<tailscale-ip>:7787` (ה-IP של המחשב ב-Tailscale; ראה בפאנל REMOTE של ה-dashboard) → "בדוק חיבור" → סמן "סנכרון אוטומטי" → שמור.
3. **סנכרון ראשוני:** במחשב, "סנכרן עכשיו" → "מקומי → לשרת". בטלפון, "סנכרן עכשיו" → "שרת → מקומי". מעכשיו הכל אוטומטי בכל שינוי.

### אפשרות ב — JSON ידני
1. בצד שיש לך התקדמות: "יצא JSON".
2. העבר את הקובץ למכשיר השני.
3. שם: "יבא JSON" → אישור.

## העלאת גזירה אישית של תמונה

100 התמונות כבר חתוכות וטעונות. אם תרצה להחליף תמונה ספציפית בגרסה חתוכה משלך:
1. עבור לשאלה שמפנה לתמונה.
2. לחץ "📷 החלף גזירה" → בחר PNG/JPG.
3. הגזירה האישית מחליפה את המוטמעת. "🗑 מחק גזירה אישית" מחזיר.

## עריכת הסברים

לחץ "עריכת/הוספת הסבר" בתחתית פאנל ההסבר. ערוך 7 שדות:
1. מסקנה רשמית
2. מה רואים
3. פירוט לפי אורות (שורה לפריט)
4. כיוון יחסי
5. סיכום פשוט
6. טעות נפוצה
7. שיטת זיכרון

סמן ☑ "מאומת" כדי להסיר את התג "דורש בדיקה ידנית".

ההסברים נשמרים ב-localStorage ועוברים בייצוא/ייבוא JSON וגם בסנכרון האוטומטי.
