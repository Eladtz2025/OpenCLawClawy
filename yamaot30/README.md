# ימאות 30 — אפליקציית לימוד

קובץ HTML יחיד, פועל מקומית, לא דורש שרת או אינטרנט אחרי הטעינה.

## מה יש בפנים

- **362 שאלות** מתוך מאגר israelsails.com (yamaot30), עם כל התשובות הנכונות.
- **PDF הסימונים הרשמי** מוטמע בתוך הקובץ כ-base64 (פתיחה דרך כפתור "פתיחת PDF הסימונים" בפאנל התמונה).
- **הסברים מפורטים מובנים** לתמונות 22, 25, 66 (לפי הנחיה ישירה — מסקנה, פירוט אורות, כיוון, טעות נפוצה, שיטת זיכרון).
- **מצבים:** סבב חכם · טעויות וניחושים · שאלות עם תמונה · מבחן 50 · מאגר חיפוש.
- **חזרה מרווחת:** סולם הפרשי 1 → 3 → 7 → 14 → 30 ימים. תשובה נכונה מקדמת, תשובה שגויה אופסה לסולם 1, סימון "ניחוש" קופץ אך לא מקדם.
- **גיבוי וייבוא JSON** להעברת התקדמות בין מחשב וטלפון.

## איך משתמשים

### במחשב
1. הורד או שמור את `yamaot30-study.html`.
2. לחץ פעמיים על הקובץ — נפתח בדפדפן (Chrome / Edge / Firefox / Safari).
3. ההתקדמות נשמרת אוטומטית ב-localStorage של הדפדפן.

### בטלפון
1. שלח את `yamaot30-study.html` לטלפון (מייל / WhatsApp / Telegram / Drive / מקל USB).
2. שמור על הטלפון, פתח עם דפדפן (Chrome / Safari).
3. (אופציונלי) הוסף ל-Home Screen (Safari → שתף → הוסף למסך הבית) להפעלה כמו אפליקציה.

### העברת התקדמות בין מחשב לטלפון
1. בצד שיש לך התקדמות: לחץ **יצא JSON** (פאנל "גיבוי וסנכרון" בימין).
2. הקובץ `yamaot30-progress-YYYY-MM-DD.json` ירד.
3. העבר אותו למכשיר השני (Drive / WhatsApp Web / מייל לעצמך).
4. בצד השני: פתח את `yamaot30-study.html`, לחץ **יבא JSON**, בחר את הקובץ, אשר.

> **שים לב:** הייבוא דורס את ההתקדמות הקיימת באותו מכשיר. גבה לפני ייבוא.

## העלאת גזירות (קרופים) של תמונות סימונים

ה-PDF הרשמי מוטמע אבל מציג עמודים שלמים. למקסימום נוחות:
1. עבור לשאלה שמפנה לתמונה (למשל #6 שמפנה לתמונות 22/25/66/2).
2. לחץ על מספר התמונה בלשונית.
3. לחץ **העלה גזירה** והעלה PNG/JPG מקומי שגזרת מה-PDF.
4. הגזירה נשמרת מקומית (לא נשלחת לשום מקום).

## עריכת הסברים

לחץ **עריכת/הוספת הסבר** בתחתית פאנל ההסבר. ערוך את 7 השדות:
1. מסקנה רשמית
2. מה רואים
3. פירוט לפי אורות (שורה לפריט)
4. כיוון יחסי
5. סיכום פשוט
6. טעות נפוצה
7. שיטת זיכרון

סמן ☑ "מאומת" כדי להסיר את התג "דורש בדיקה ידנית".

## מה עדיין דורש עבודה ידנית

- **גזירות PNG לכל 57 התמונות** — כרגע מוצגת הצעה לפתוח את ה-PDF. אם תיצור 57 גזירות PNG נפרדות אפשר להטמיע אותן (אני יכול להוסיף flow לעדכון מסיבי).
- **הסברים מפורטים לכ-54 התמונות הנותרות** (לא מתוך ה-3 המוטמעות).
- **קטגוריות** — נקבעו אוטומטית לפי מילות מפתח. עבור 362 שאלות זה לא מושלם — ערוך לפי הצורך.
- **פסקת זיהוי "כל התשובות נכונות"** וצורות שאלה אחרות — האפליקציה מטפלת בהן כתשובה רגילה (אם המקור מסמן ד׳ כנכונה, זה הציון הנכון).
